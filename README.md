# Matrix Operations Tool

A simple Python project that performs basic matrix operations using the NumPy library. The application provides an interactive menu-driven interface for performing common matrix calculations.

## Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Matrix Transpose
* Determinant Calculation
* Matrix Inverse

## Technologies Used

* Python
* NumPy

## Installation

Install the required library:

```bash
pip install numpy
```

## Run the Project

```bash
python matrix_operations.py
```
